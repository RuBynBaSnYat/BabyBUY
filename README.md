# BabyBUY
BabyAPP
👶 BabyBuy – Baby Shopping Organizer App
BabyBuy is a user-friendly Android app built with Kotlin using Android Studio that helps expecting parents prepare for their baby’s arrival by organizing and managing a personalized shopping list for baby items. The app allows users to add, edit, delete, and share items through SMS, ensuring tasks are well-delegated among family and friends.

📲 About the App
Expecting a baby is a beautiful journey—but it can also be stressful when it comes to preparation. BabyBuy makes it easier by giving parents a smart checklist they can access, update, and share on the go.

🔑 Core Features
🔐 User Authentication: Simple registration & login

📝 Add/Edit/Delete Items: Manage shopping lists with full CRUD functionality

📸 Image Uploads: Add product photos for better reference

📍 Location Tagging: Add map-based item locations

✅ Purchase Status: Mark items as purchased

📤 SMS Sharing: Share shopping tasks with family and friends

🔍 Search Functionality: Find items in your list easily

👤 Profile & Logout: View user profile and securely log out

🛠 Tech Stack
Language: Kotlin

IDE: Android Studio

Database: RoomDB (SQLite abstraction)

Architecture: Native Android (Single Activity + multiple fragments)

Tools Used: Material Design Components, SMS Manager API, Google Maps Integration (for location)
